The Problem Statement

The USDA (United States Department of Agriculture) works to provide leadership and management on subjects such as food, agriculture, natural resources, rural development and nutrition. The agency's website creates struggle for usability due to text-heavy sections and overwhelming page layouts. With so much important information the agency has to provide, the solution is to redesign the main user interface navigation and website flow. The goal is to create a more minimalistic and appealing layout design while condensing the text and categories throughout the main navigation.

Research & Analysis

Research began by analyzing the user interface to get an idea of the user path/journey. Afterwards, testing and annotation of the navigation took place along with creating a sitemap. Finally, wireframing and incorporating user interface design occurred followed by testing of the new design. After some further iteration and application of user interface elements, a design layout was finalized.

The process began by creating a proto-persona of a perceived user of the USDA website. Creating this was able to kick-off further research for the rest of the project.

Wireframes were created in order to build off of the sitemap created. I was able to define the problems and synthesize with use of card sorting and site mapping.

Iterations

Both mobile and desktop views gained positive results about the organization of the new layout and how easy it is to navigate through. The users appreciated that there is less text and clutter from too many category options and how visually organized the redesign looks.

Low/mid fidelity wireframes for mobile and desktop were iterated upon in order to create a clickable prototype and get to a high fidelity result. User interface style guide was created in order to see and organize style choices, buttons, etc. used throughout the redesign.
